Model Overview

The flood prediction system was trained using multiple machine learning algorithms on historical rainfall and meteorological datasets. XGBoost achieved the highest prediction accuracy of 96.55% and was selected as the final deployment model.

The model evaluates ten environmental parameters including temperature, humidity, cloud cover, annual rainfall and seasonal rainfall before predicting flood susceptibility.
